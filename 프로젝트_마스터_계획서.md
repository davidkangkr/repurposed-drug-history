## 4. 책 내용 작성 원칙 (프롬프트 필수 포함)

```
=== 법적 안전 원칙 ===
- "효과가 있다" "치료된다" "암이 낫는다"
  → 절대 단정 금지
- "주목받고 있다" "가능성이 연구되고 있다"
  "관심을 받고 있다" → 허용
- 구충제, 무좀약 등 직접 비하적 표현 자제
  → "오래된 약" "용도변경된 약"으로 표현
- 특정 제품명 광고성 언급 금지
- 복용을 직접 권유하는 표현 금지
  → "담당 의사와 상의하세요" 항상 포함

=== 내용 방향 원칙 ===
- 항암 효과 중심이 아닌
  약의 근본 원리 중심으로 서술
- 흐름: 탄생 배경 → 원래 작동 원리
  → "이 원리가 암세포에도?" → 연구 현황
- 독자가 스스로 납득하도록 유도
  → 결론을 강요하지 말 것
- 과장 금지, 한계와 불확실성 솔직하게
- 세계 환자들의 자발적 관심 맥락 포함

=== 언어 원칙 ===
- 한자어, 영어 의학용어 단독 사용 금지
  → 반드시 쉬운 설명 병기
- 순우리말 우선 사용
- MZ세대부터 70대까지 이해 가능한 언어
  (단, 각 권 페르소나에 맞게 조정)
- 돌보는 이: 30자 이내 짧은 문장
- 싸우는 이: 자연스러운 이야기체
- 파고드는 이: 전문용어 허용, 출처 명시

=== 톤 원칙 ===
- 희망적이되 균형잡힌 시각
- 독자를 가르치려 하지 말 것
- 이웃이 이야기 들려주듯 따뜻하게
- 커뮤니티(히포크라테스의 친구들)의
  공부하는 자세 반영
- 보고서 형식 절대 금지
  → 이야기 형식으로

=== 저자 원칙 ===
- "저는 의사가 아닙니다" 명시
- "2020년부터 공부한 일반인"임을 밝힘
- 커뮤니티에서 함께 공부한 내용임을 밝힘
- 기존 치료를 대체하는 것이 아님을 명시
```
