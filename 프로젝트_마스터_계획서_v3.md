## 0. 이 책의 핵심 의도

용도변경된 약들은 누구나
원래 용도로 알고 있습니다.
구충제, 당뇨약, 항생제, 향신료...

이 책은 그 약들이
원래 목적 외에 왜 암세포에
가능성을 가지는지
그 원리를 알려줍니다.

모르면 막연하게 시도합니다.
알면 원리를 이해하고 판단해서 시도합니다.
이것이 이 책에서 말하는 "안다"의 의미입니다.

세계적으로, 한국에서도
이 약들로 호전된 사례들이
실제로 나오고 있습니다.
저자(꿈많은사람2)는
   히포크라테스의 친구들
   (cafe.naver.com/friendsofhippocrates)에서
   환자 주도형 암 임상에 관심을 갖고
   함께 공부하고 참여하고 있습니다.

이 책은:
- "쓰지 마라"가 아닙니다
- "원리를 알고 가능성을 이해하고 시도하자"입니다
- 독자는 이미 관심이 있는 사람입니다
- 막는 것이 아니라 올바르게 안내합니다

## 4. 챕터 작성 공통 원칙

```
=== 내용 원칙 ===
- 내용과 사실은 3권 모두 동일
- 페르소나는 독자가 들어오는 "문"만 바꿈
- 감정은 문을 여는 역할만, 사실 호도 금지
- 팩트체크 단계 생략 (작성 후 검토는 별도)
- 참고문헌은 조사 파일에서 유지
- 각 권 스타일은 샘플 파일 직접 참조

=== 금지 (챕터 본문 작성 시) ===
- 볼드체(**) 사용 금지
- 번호 나열(첫째, 둘째) 금지
- 보고서 형식 금지
- 효능 단정 표현 금지 ("효과가 있다")
- 기존 치료 대체 권유 금지
- 복용 직접 권유 금지
- 특정 제품명 광고성 언급 금지
- 가능성을 닫는 표현 금지
- "하지만 여기서 멈춰야 합니다" 금지
- "과장된 정보가 많습니다" 식 불신 조장 금지
- 독자를 겁주는 톤 금지

=== 편집 단계에서 허용 및 추가 ===
- 볼드체 (중요 포인트 강조)
- 번호 나열 (목차, 순서)
- 보고서 형식 (참고문헌 등)
- 의학 용어 원어 병기:
  1권 박명순: 원어 생략, 쉬운 설명만
  2권 설숙자: 필요시 원어 병기
  3권 최연옥: 원어 항상 병기
  예시: 펜벤다졸 (Fenbendazole)
        미세소관 (microtubule, 微細管)
        세포자멸사 (apoptosis, 細胞自滅死)

=== 허용 표현 ===
- "주목받고 있다", "가능성이 연구되고 있다"
- "관심을 받고 있다", "검토되고 있다"
- 구충제/무좀약 등 직접 비하적 표현 자제
  → "오래된 약", "용도변경된 약"으로 표현
- "담당 의사와 상의" 항상 포함

=== 언어 ===
- 한자어, 영어 의학용어 단독 사용 금지
  → 반드시 쉬운 설명 병기
- 순우리말 우선 사용
- 각 권 페르소나에 맞게 난이도 조절
```

## 6. 챕터 구성 (각 약물 공통)

```
소제목 1: 이 약은 어디서 왔나?
→ 탄생 시대 배경
→ 개발한 회사/연구자 이야기
→ 왜 이 약이 만들어졌는지

소제목 2: 어떻게 작동하나?
→ 원래 작용 원리 (일상 비유로)
→ 기생충/균/바이러스를 어떻게 잡는지
→ 충분히 이해시키기

소제목 3: 그런데... 암세포도?
→ 원리의 자연스러운 연결
→ 과학자들의 발견 과정
→ 흥미로운 사례 (조 티펜스 등)

소제목 4: 지금 연구는 어디까지?
→ 세포실험/동물실험 결과
→ 임상시험 현황 (솔직하게)
→ 한계 명시

소제목 5: 여러분이 알아야 할 것
→ 주의사항
→ 담당 의사와 상의
→ 커뮤니티 현황 (해당시)
```
