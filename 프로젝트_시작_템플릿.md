# 새 프로젝트 시작 템플릿 (범용)

## 프로젝트 기본 정보

```
프로젝트명:
목적:
예상 기간:
주요 산출물:
참여 도구:
```

---

## 1. 마스터 계획서 구성

### 1.1 프로젝트 핵심 의도
프로젝트의 목적과 방향을 한 문장으로 정의합니다.

### 1.2 작업 단위/산출물 구성
프로젝트의 작업을 단위별로 나누고 각 산출물을 정의합니다.

### 1.3 사용자/대상 설정
산출물을 사용하거나 영향을 받는 대상을 정의합니다.
대상에 따라 톤과 난이도를 조절합니다.

---

## 2. 환경 설정

### 2.1 필수 파일
```
CLAUDE.md          ← 프로젝트 핵심 원칙 (항상 필요)
마스터_계획서.md    ← 전체 작업 계획 (중간 규모 이상)
```

### 2.2 Harness 선택 기준
| 작업 유형 | 선택 |
|-----------|------|
| 반복 작업 자동화 | 해당 도메인 Harness |
| 리서치/조사 | documentary-research |
| 편집/교정 | book-publishing |
| 단순 수정 | 일반 Claude Code |

### 2.3 기타 도구
- 웹 검색: WebSearch / WebFetch
- 논문 검색: PubMed / Google Scholar
- 이미지 생성: Gemini / DALL-E / Midjourney

---

## 3. 작업 파일 구조 (참고)

```
프로젝트 루트\
├── CLAUDE.md                ← 핵심 원칙
├── 마스터_계획서.md          ← 전체 계획
├── 프로젝트_진행기록.md      ← 진행 상황
├── src\ 또는 작업폴더\
│   └── (산출물 파일들)
└── _workspace\              ← 임시 작업물
```

---

## 4. 원칙 정의

### 금지
- 결과를 단정하는 표현
- 대상자를 무시하거나 겁주는 톤
- 신뢰도를 해치는 과장

### 허용
- "검토되고 있다", "가능성이 있다"
- 출처 명시 (해당 시)
- 담당자/전문가와 상의 안내

---

## 5. 규모별 템플릿 적용

| 규모 | 예상 기간 | 필요한 파일 |
|------|-----------|-----------|
| 단순 | 하루 | CLAUDE.md만 |
| 중간 | 며칠~몇주 | CLAUDE.md + 마스터 계획서 |
| 대형 | 몇달 | 4개 파일 모두 |

---

이 템플릿은 책 출판, 웹개발, 데이터 분석, 마케팅 등
모든 프로젝트에 적용 가능합니다.
